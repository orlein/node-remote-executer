# HOW TO RUN THIS

1. npm install

2. npm i -g pm2

3. pm2 start index.js

4. 현재 컴퓨터에서 PORT 3000으로 붙은거임

  > 바꾸고싶으면 index.js 맨아래 port 바꾸면됨

5. <주소>:<PORT>로 알아서 들어가서 커맨드라인 입력하고 엔터치고 하면 알아서 실행됨
